Guia rapida

Flujo esencial de Git + GitHub

Una secuencia clara para crear tu proyecto, versionarlo y subirlo a GitHub sin perder el ritmo.

6 pasos CLI first Deploy mental model
  1. 01
    Inicio

    Inicializar proyecto

    mkdir proyecto-git
    cd proyecto-git
    git init
  2. 02
    Commit

    Primer commit local

    git add .
    git commit -m "proyecto base"
  3. 03
    Web

    Crear repo en GitHub

    Crea el repositorio remoto desde la web y copia la URL para el enlace local.

  4. 04
    Remote

    Vincular con GitHub

    git remote add origin URL_DEL_REPO_REMOTO
  5. 05
    Push

    Subir a remoto

    git push origin master
  6. 06
    Iterar

    Actualizar cambios

    git add .
    git commit -m "nuevo cambio"
    git push origin master
